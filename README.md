# pikastat-yaui

Very secret UI

## How to build?

0. (Optional) make your fork
1. Clone

    `git clone https://github.com/DevAlone/pikastat-yaui.git`

2. Create env

    `cd pikastat-yaui`

    `python3 -m venv env`

3. Activate it

    `source env/bin/activate`

4. Install deps

    `pip3 install -r requirements.txt`

5. Run

    `python3 main.py`
